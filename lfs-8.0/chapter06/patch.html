6.62. Patch-2.7.5

Patch 软件包包含一个通过打“补丁” 来创建或修改文件的程序，补丁文件通常由 diff 程序生成。

大概编译时间： 0.2 SBU
所需磁盘空间： 11 MB

6.62.1. 安装 Patch

准备编译 Patch ： 

./configure --prefix=/usr

编译软件包： 

make

测试编译结果，执行： 

make check

安装软件包： 

make install

6.62.2. Patch 的内容

安装的程序： patch

简介

patch

根据补丁文件来修改文件 [补丁文件通常是由 diff 程序创建的差异列表。 patch 通过对原始文件应用这些差异创建打补丁后的版本。]
