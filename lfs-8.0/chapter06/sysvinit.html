6.64. Sysvinit-2.88dsf

Sysvinit软件包包含了控制系统启动、运行和关机的程序。

预计编译时间； 少于 0.1 SBU
所需磁盘空间； 1.1 MB

6.64.1. 安装Sysvinit软件包

首先，安装一个补丁，用来移除其他软件包安装的几个程序，清除信息，解决编译警告； 

patch -Np1 -i ../sysvinit-2.88dsf-consolidated-1.patch

编译软件包； 

make -C src

这个软件包没有附带测试程序。

安装软件包； 

make -C src install

6.64.2. Sysvinit包含的内容

安装的程序； bootlogd, fstab-decode, halt, init, killall5, poweroff (link to halt), reboot (link to halt), runlevel, shutdown, and telinit (link to init)

简述

bootlogd

把启动信息记录到一个日志文件里

fstab-decode

运行一个带有fstab-encoded参数的命令

halt

系统运行级别为0，halt告诉内核关闭系统，否则等同于调用 shutdown 带 -h 参数； 将系统将要关机的事件记录到 /var/log/wtmp 文件里。

init

内核初始化完硬件之后，接管启动进程和启动在配置文件中所有指定程序的第一个进程。

killall5

发送信号给所有进程，除非进程在自己会话中并且不会杀死它的父进程。

poweroff

告诉内核关闭系统并且关掉计算机 (见 halt)

reboot

告诉内核重启系统 (见 halt)

runlevel

报告上次和当前的运行级别，比如被记录在 /var/run/utmp文件中的上次运行级别记录。

shutdown

以一种安全的方式关机，向所有进程发信号并且通知所有登录系统的用户。

telinit

告知 init 将系统放置在特定的运行级别上。
