6.61. Make-4.2.1

Make 软件包包含用于编译软件包的程序

大概编译时间： 0.5 SBU
所需磁盘空间： 12.6 MB

6.61.1. 安装 Make

准备编译 Make ： 

./configure --prefix=/usr

编译软件包： 

make

测试编译结果，执行： 

make check

安装软件包： 

make install

6.61.2. Make 的内容

安装的程序： make

简介

make

自动判断一个软件包的哪些部分需要（重新）编译，然后执行相关命令
