6.51. Diffutils-3.5

Diffutils 是实现显示文件和目录差异的软件包

大概编译时间: 0.4 SBU
需要磁盘空间: 30 MB

6.51.1. 安装 Diffutils

首先修复文件安装区域文件: 

sed -i 's:= @mkdir_p@:= /bin/mkdir -p:' po/Makefile.in.in

配置 Diffutils: 

./configure --prefix=/usr

编译软件包: 

make

安装前检查: 

make check

安装: 

make install

6.51.2. Diffutils 软件包内容

安装的程序: cmp, diff, diff3, 和 sdiff

简要介绍

cmp

比较两个文件并报告差异

diff

比较两个文件或目录并报告文件中的差异行

diff3

逐行比较三个文件

sdiff

合并两个文件，并且交互输出结果
