6.15. MPFR-3.1.5

MPFR 软件包包含多精度数学函数库。

预计构建时间: 0.8 SBU
所需磁盘空间: 45 MB

6.15.1. 安装 MPFR

准备编译 MPFR: 

./configure --prefix=/usr        \
            --disable-static     \
            --enable-thread-safe \
            --docdir=/usr/share/doc/mpfr-3.1.5

编译软件包并生成 HTML 文档: 

make
make html

重要

本节中 MPFR 的测试套件至关重要，任何情况下都不能跳过。

测试编译结果确保所有的测试都通过: 

make check

安装软件包和文档: 

make install
make install-html

6.15.2. MPFR 软件包内容

安装的库: libmpfr.so
安装的文件夹: /usr/share/doc/mpfr-3.1.5

简要介绍

libmpfr

包含多精度数学函数
