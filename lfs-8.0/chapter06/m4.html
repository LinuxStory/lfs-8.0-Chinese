6.28. M4-1.4.18

M4 软件包包含一个宏处理器。

预计构建时间: 0.4 SBU
所需磁盘空间: 31 MB

6.28.1. 安装 M4

准备编译 M4 : 

./configure --prefix=/usr

编译该软件包: 

make

测试编译过程，运行: 

make check

安装该软件包: 

make install

6.28.2. M4 的内容

安装的程序: m4

简要描述

m4

复制给定的文件，同时扩展它们包含的宏[这些宏是内置的或用户定义的，可以携带任何数量的参数。 除了执行宏扩展, m4 还内置了很多功能，包括：命名文件、运行Unix命令、执行整数运算、操作文本、递归等等。m4 程序能被用作编译器的前端或是作为宏处理器使用。]
