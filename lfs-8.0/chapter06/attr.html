6.21. Attr-2.4.47

attr软件包包含了一些实用程序用于管理文件系统对象上的扩展属性。

预计编译时间: 少于 0.1 SBU
所需磁盘空间: 3.3 MB

6.21.1. 安装 Attr

修改文档目录使其成为一个版本化的目录: 

sed -i -e 's|/@pkg_name@|&-@pkg_version@|' include/builddefs.in

将手册页面的安装排除在帮助页面 包安装过程之外: 

sed -i -e "/SUBDIRS/s|man[25]||g" man/Makefile

为 Attr 的编译做准备: 

./configure --prefix=/usr \
            --bindir=/bin \
            --disable-static

编译这个软件包: 

make

测试需要运行在支持扩展属性的文件系统上，如： ext2, ext3, 或 ext4 文件系统。已知该测试会在同时运行多个测试(-j 选项大于1)时出现失败。要测试结果，请执行以下操作： 

make -j1 tests root-tests

安装这个软件包: 

make install install-dev install-lib
chmod -v 755 /usr/lib/libattr.so

这个共享库需要被移动到 /lib, 并且需要在/usr/lib目录下生成文件 .so: 

mv -v /usr/lib/libattr.so.* /lib
ln -sfv ../../lib/$(readlink /usr/lib/libattr.so) /usr/lib/libattr.so

6.21.2. Attr 的内容

安装的程序: attr, getfattr, and setattr
安装的库: libattr.so
安装的目录: /usr/include/attr 和 /usr/share/doc/attr-2.4.47

简要描述

attr

文件系统对象上的扩展属性

getfattr

获取文件对象的扩展属性

setattr

设置文件对象的扩展属性

libattr

包含用于操作扩展属性的库函数
