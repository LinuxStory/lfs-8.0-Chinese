6.8. Man-pages-4.09

The Man-pages package contains over 2,200 man pages. Man-pages 软件包里包含了超过 2,200 份的手册页。

预计构建时间： 少于 0.1 SBU
所需磁盘空间： 27 MB

6.8.1. 安装 Man-pages

运行下面的命令安装 Man-pages： 

make install

安装的文件： 各种手册页
