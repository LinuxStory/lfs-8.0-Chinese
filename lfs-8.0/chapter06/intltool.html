6.42. Intltool-0.51.0

Intltool 是用与将可翻译的文本从源文件中提取出来的国际化工具。

预计构建时间: 少于 0.1 SBU
需要磁盘空间: 1.5 MB

6.42.1. Intltool 安装过程

首先修复由于 perl-5.22 及之后版本导致的一个警告: 

sed -i 's:\\\${:\\\$\\{:' intltool-update.in

准备编译 Intltool: 

./configure --prefix=/usr

编译软件包: 

make

测试编译结果，输入: 

make check

安装软件包: 

make install
install -v -Dm644 doc/I18N-HOWTO /usr/share/doc/intltool-0.51.0/I18N-HOWTO

6.42.2. Intltool 的内容

安装的程序: intltool-extract, intltool-merge, intltool-prepare, intltool-update, and intltoolize
安装的目录: /usr/share/doc/intltool-0.51.0 and /usr/share/intltool

简要介绍

intltoolize

准备一个软件包以使用 intltool

intltool-extract

生成可以被 gettext 读取的头文件

intltool-merge

将已翻译的字符串合并为不同的文件类型

intltool-prepare

更新 pot 文件并将他们与翻译文件合并

intltool-update

更新 po 模板文件并将他们与翻译合并
