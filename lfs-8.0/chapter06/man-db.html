6.67. Man-DB-2.7.6.1

Man-DB 软件包包含用于查找和浏览 man 页面的程序。

构建约需时间： 0.4 SBU
需要磁盘空间： 30 MB

6.67.1. 安装 Man-DB

准备编译 Man-DB： 

./configure --prefix=/usr                        \
            --docdir=/usr/share/doc/man-db-2.7.6.1 \
            --sysconfdir=/etc                    \
            --disable-setuid                     \
            --enable-cache-owner=bin             \
            --with-browser=/usr/bin/lynx         \
            --with-vgrind=/usr/bin/vgrind        \
            --with-grap=/usr/bin/grap            \
            --with-systemdtmpfilesdir=

配置选项的含义：

--disable-setuid

为 man 用户禁止 man 程序调用 setuid 。

--enable-cache-owner=bin

改变系统缓存文件拥有者为用户 bin 。

--with-...

这三个参数用来设置一些默认程序。 lynx 是一个基于文本的网络的网页浏览器（见 BLFS 安装指引），vgrind 将程序源码转化为 Groff 输入，grap 在 Groff 文档排版图时非常有用。vgrind 和 grap 程序在浏览手册时通常不需要。它们通常不是 LFS 和 BFLS 的一部分，如果你需要的话可以在安装完 LFS 后自行安装。

编译软件包： 

make

测试结果： 

make check

安装软件包： 

make install

6.67.3. Man-DB 内容

安装的程序： accessdb, apropos (链接到 whatis), catman, lexgrog, man, mandb, manpath, and whatis
安装的库： libman.so and libmandb.so
安装目录： /usr/lib/man-db, /usr/lib/tmpfiles.d, /usr/libexec/man-db, 和 /usr/share/doc/man-db-2.7.6.1

简要描述

accessdb

将 whatis 数据库中的内容转储为可阅读格式

apropos

搜索 whatis 数据库并显示含所查找字符串的系统命令的简述

catman

创建或更新预格式化的手册页

lexgrog

显示所提供的手册页的一行概要

man

格式化显示请求的手册页

mandb

创建或更新 whatis 数据库

manpath

显示 $MANPATH 或（如果未设置 $MANPATH）基于用户环境中 man.conf 中的合适的搜索路径的内容

whatis

搜索 whatis 数据库并显示含所查找独立关键词的系统命令的简述

libman

包含 man命令的运行时支持

libmandb

包含 man命令的运行时支持
