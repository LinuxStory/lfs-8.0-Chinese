6.24. Sed-4.4

Sed 软件包包含了一个流编辑器。

预计构建时间: 0.3 SBU
所需磁盘空间: 25 MB

6.24.1. 安装 Sed

首先修正一个在 LFS 环境中的问题并且去除一个失效的测试: 

sed -i 's/usr/tools/'       build-aux/help2man
sed -i 's/panic-tests.sh//' Makefile.in

准备编译 Sed 软件包: 

./configure --prefix=/usr --bindir=/bin

编译该软件包并生成HTML文档: 

make
make html

编译测试，如下: 

make check

安装该软件包和对应文档: 

make install
install -d -m755           /usr/share/doc/sed-4.4
install -m644 doc/sed.html /usr/share/doc/sed-4.4

6.24.2. Sed 的内容

安装的程序: sed
安装的目录: /usr/share/doc/sed-4.4

简要描述

sed

单向过滤和转换文本文件
