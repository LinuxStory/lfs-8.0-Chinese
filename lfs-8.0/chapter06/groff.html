6.54. Groff-1.22.3

Groff 软件包包含用于处理和格式化文本的程序。

大概编译时间： 0.5 SBU
所需磁盘空间： 82 MB

6.54.1. 安装 Groff

Groff 希望环境变量 PAGE 包含默认的页面大小。对于美国的用户，设置为 PAGE=letter 较为恰当。 其他地区设置为 PAGE=A4 更合适。 默认的页面大小虽然是在编译时设置的，但仍可通过 echo “A4” 或者 “letter” 到 /etc/papersize 文件覆盖设置。

准备编译 Groff ： 

PAGE=<paper_size> ./configure --prefix=/usr

编译软件包： 

make

本软件包不带测试套件。

安装软件包： 

make install

6.54.2. Groff 的内容

安装的程序： addftinfo, afmtodit, chem, eqn, eqn2graph, gdiffmk, glilypond, gperl, gpinyin, grap2graph, grn, grodvi, groff, groffer, grog, grolbp, grolj4, gropdf, grops, grotty, hpftodit, indxbib, lkbib, lookbib, mmroff, neqn, nroff, pdfmom, pdfroff, pfbtops, pic, pic2graph, post-grohtml, preconv, pre-grohtml, refer, roff2dvi, roff2html, roff2pdf, roff2ps, roff2text, roff2x, soelim, tbl, tfmtodit, and troff
安装的目录： /usr/lib/groff and /usr/share/doc/groff-1.22.3, /usr/share/groff

简介

addftinfo

读取 troff 字体文件并添加一些用于 groff 系统的额外字体信息

afmtodit

创建用于和 groff 以及 grops 一起使用的字体文件

chem

Groff 用于生成化学结构图的预处理器

eqn

将嵌在 troff 输入文件中的等式的描述编译为 troff 能理解的命令

eqn2graph

将 troff EQN（方程式）转换为裁剪图像

gdiffmk

标记 groff/nroff/troff 文件的差异

glilypond

将 lilypond 语言编写的乐谱转换为 groff 语言

gperl

groff 的预处理器，支持向 groff 文件中添加 perl 代码

gpinyin

groff 的预处理器，支持向 groff 文件中添加中文语言的欧式拼音

grap2graph

将 grap 图转换为裁剪位图图像

grn

用于 gremlin 文件的 groff 预处理器

grodvi

处理 TeX dvi 格式的 groff 驱动

groff

一种 groff 文档格式化系统的前端； 通常它会运行 troff 程序以及一个适用于选定设备的后处理器

groffer

在 X 和 tty 终端上显示 groff 文件和 man 页面

grog

通过读取文件来猜测 groff 的选项 -e, -man, -me, -mm, -ms, -p, -s, and -t 中哪些是打印文件所需的，并报告包含这些选项的 groff 命令

grolbp

用于佳能 CAPSL 打印机（LBP-4 和 LBP-8 系列激光打印机）的 groff 驱动

grolj4

生成适用于 HP LaserJet 4 打印机的 PCL5 格式输出的 groff 驱动

gropdf

将 GNU troff 的输出转换为 PDF

grops

将 GNU troff 的输出转换为 PostScript

grotty

将 GNU troff 的输出转换为适用于打字机设备的格式

hpftodit

从 HP 标签字体格式文件创建和 groff -Tlj4 一起使用的字体文件

indxbib

根据指定文件创建用来和 refer， lookbib， 以及 lkbib 一起使用的文献数据库的倒排索引

lkbib

在文献数据库中搜索包含指定关键字的引用并报告任何找到的引用

lookbib

在标准错误中打印提示（除非标准输入不是终端），从标准输入中读入 包含一组关键字的行，在指定文件中搜索文献数据库中包含那些关键字的引用， 将找到的引用打印到标准输出，并重复此过程直到输入结束

mmroff

groff 的简单预处理器

neqn

格式化以美国标准信息交换码（ASCII）输出的方程

nroff

groff 模拟 nroff 的脚本

pdfmom

groff 的一种封装，用于将 mom 宏格式化过的文件生成 PDF 文档

pdfroff

用 groff 生成 PDF 文档

pfbtops

将 .pfb 格式的 PostScript 字体转换为 ASCII

pic

将嵌入在 troff 或 TeX 输入文件中的图片描述编译为可被 TeX 或 troff 理解的命令

pic2graph

将 PIC 图像转换为裁剪图像

post-grohtml

将 GNU troff 的输出转换为 HTML

preconv

将输入文件的编码转换为 GNU troff 能理解的形式

pre-grohtml

将 GNU troff 的输出转换为 HTML

refer

将文件的内容复制到标准输出， 其中 .[ 和 .] 之间的行解释为引用， .R1 和 .R2 之间的行解释为处理引用的命令

roff2dvi

将 roff 文件转换为 DVI 格式

roff2html

将 roff 文件转换为 HTML 格式

roff2pdf

将 roff 文件转换为 PDF

roff2ps

将 roff 文件转换为 ps 文件

roff2text

将 roff 文件转换为文本文件

roff2x

将 roff 文件转换为其他格式

soelim

读入文件并用相应的 file 内容替换 .so file 格式的行

tbl

将嵌入 troff 输入文件中的表格的描述编译为 troff 理解的命令

tfmtodit

创建和 groff -Tdvi 一起使用的字体文件

troff

高度兼容 Unix troff；通常应该被 groff 命令所调用，而后者会以恰当的选项运行预处理器和后处理器
