6.38. Expat-2.2.0

Expat 软件包，包含一个用于解析 XML 的面向流的 C 库。

预计构建时间： 少于 0.1 SBU
所需磁盘空间： 6.1 MB

6.38.1. 安装 Expat

准备编译 Expat： 

./configure --prefix=/usr --disable-static

编译软件包： 

make

要测试结果，运行： 

make check

安装软件包： 

make install

如果需要的话，安装帮助文档： 

install -v -dm755 /usr/share/doc/expat-2.2.0
install -v -m644 doc/*.{html,png,css} /usr/share/doc/expat-2.2.0

6.38.2. Expat 包含内容

安装的程序： xmlwf
安装的库： libexpat.so
安装目录： /usr/share/doc/expat-2.2.0

简介

xmlwf

一个检测 XML 文档格式是否完整的非验证工具

libexpat

包含用于解析 XML 的 API 函数
