6.68. Tar-1.29

Tar 软件包包含一个归档程序。

构建约需时间： 3.2 SBU
需要磁盘空间： 40 MB

6.68.1. 安装 Tar

准备编译 Tar： 

FORCE_UNSAFE_CONFIGURE=1  \
./configure --prefix=/usr \
            --bindir=/bin

配置选项的含义：

FORCE_UNSAFE_CONFIGURE=1

强制以 root 用户运行 mknod 测试。通常认为以 root 用户运行这个测试是危险的，但是在未构建完的系统上这样做是可以的。

编译软件包： 

make

用以下命令测试结果（约 1 SBU）： 

make check

安装软件包： 

make install
make -C doc install-html docdir=/usr/share/doc/tar-1.29

6.68.2. Tar 的内容

安装的程序： tar
安装目录： /usr/share/doc/tar-1.29

简要描述

tar

创建，解压，显示归档文件内容，归档文件也被称为 tar 包。
