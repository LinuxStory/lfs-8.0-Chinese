6.29. Bison-3.0.4

Bison 软件包包含解析器生成器.

预计编译时间: 0.3 SBU
所需磁盘空间: 32 MB

6.29.1. 安装 Bison

准备编译 Bison: 

./configure --prefix=/usr --docdir=/usr/share/doc/bison-3.0.4

编译该软件包: 

make

关于编译检查，会存在一个 bison 包和 flex 包的循环依赖。如果需要编译检查，先安装 flex，再进行 bison 包的编译检查make check.

安装该软件包: 

make install

6.29.2. Bison 的内容

安装的程序: bison and yacc
安装的库: liby.a
安装的程序: /usr/share/bison

简要描述

bison

从一系列规则中生成用于分析文本文件结构的程序; Bison是Yacc（另一个编译器编译器）的替代品

yacc

一个 bison的包装器, 意味着其他程序仍能够调用 yacc 而不是 bison 命令; 实际相当于调用了 bison -y

liby

该 Yacc 类库用于实现兼容于 Yacc 的yyerror 和 main 功能; 该类库并不常用，但是 POSIX 标准需要。
