6.39. Inetutils-1.9.4

Inetutils 软件包，包含基本网络程序。

预计构建时间： 0.4 SBU
所需磁盘空间： 27 MB

6.39.1. 安装 Inetutils

准备编译 Inetutils： 

./configure --prefix=/usr        \
            --localstatedir=/var \
            --disable-logger     \
            --disable-whois      \
            --disable-rcp        \
            --disable-rexec      \
            --disable-rlogin     \
            --disable-rsh        \
            --disable-servers

配置选项的含义：

--disable-logger

此选项防止 Inetutils 安装 logger 程序，脚本使用此程序向系统日志守护进程传递消息。不要安装它，因为 Util-linux 会安装一个较新的版本。

--disable-whois

此选项禁ommand>whois 客户端，在 BFLS 中会有指导。书

--disable-r*

这些参数禁用了废弃程序的构建，由于安全问题而不该使用它们。它们所提供的函数可以由 BLFS 书中的 openssh 软件包提供。

--disable-servers

此参数禁用了多个网络服务程序的安装，它们是 Inetutils 软件包的一部分。这些服务程序被认为不适用于基础的 LFS 系统。有些程序本来就不安全，只在可信网络中才能被认为是安全的。注意，它们很多都有更好的替代品。

编译软件包： 

make

要测试结果，运行： 

make check

安装软件包： 

make install

移动一些程序，这样在 /usr 无法访问时它们仍然可用。 

mv -v /usr/bin/{hostname,ping,ping6,traceroute} /bin
mv -v /usr/bin/ifconfig /sbin

6.39.2. Inetutils 包含内容

安装的程序： dnsdomainname, ftp, ifconfig, hostname, ping, ping6, talk, telnet, tftp, 和 traceroute

简介

dnsdomainname

显示系统的 DNS 域名

ftp

文件传输协议程序

hostname

报告或者设置主机名称

ifconfig

管理网络接口

ping

发送请求响应包，并报告响应用时

ping6

ping 用于 IPv6 网络的版本。

talk

用来和另一个用户交谈

telnet

TELNET 协议的一个接口

tftp

一个文件传输小程序

traceroute

追踪当前主机发送到网络上另一台主机的数据包的路径，显示沿路的所有中间跃点（网关）。
