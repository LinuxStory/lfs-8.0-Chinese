6.55. GRUB-2.02~beta3

GRUB 软件包包含多重启动管理器（GRand Unified Bootloader）。

大概编译时间： 0.8 SBU
所需磁盘空间： 142 MB

6.55.1. 安装 GRUB

准备编译 GRUB ： 

./configure --prefix=/usr          \
            --sbindir=/sbin        \
            --sysconfdir=/etc      \
            --disable-efiemu       \
            --disable-werror

新配置选项的含义：

--disable-werror

允许在新近的 Flex 版本引发的警告的情况下仍然完成构建。

--disable-efiemu

这个选项通过通用 LFS 不需要的功能和测试程序以实现最小化构建。

编译软件包： 

make

本软件吧不包含测试套件。

安装软件包： 

make install

 8.4章节, “使用 GRUB 设置启动过程”将会讨论使用 GRUB 启动你的 LSF 系统。

6.55.2. GRUB 的内容

安装的程序： grub-bios-setup, grub-editenv, grub-file, grub-fstest, grub-glue-efi, grub-install, grub-kbdcomp, grub-macbless, grub-menulst2cfg, grub-mkconfig, grub-mkimage, grub-mklayout, grub-mknetdir, grub-mkpasswd-pbkdf2, grub-mkrelpath, grub-mkrescue, grub-mkstandalone, grub-ofpathname, grub-probe, grub-reboot, grub-render-label, grub-script-check, grub-set-default, grub-sparc64-setup, and grub-syslinux2cfg
安装的目录： /usr/lib/grub, /etc/grub.d, /usr/share/grub, 和 boot/grub （当 grub-install 第一次被执行时）

简介

grub-bios-setup

grub-install 的一个帮助程序

grub-editenv

编辑环境块的工具

grub-file

检查 FILE 是不是指定类型。

grub-fstest

调试文件系统驱动的工具

grub-glue-efi

处理 ia32 和 amd64 EFI 镜像并根据 Apple 格式将它们拼合。

grub-install

将 GRUB 安装到你的驱动器

grub-kbdcomp

将 xkb 布局转换成一种可被 GRUB 识别的形式的脚本

grub-macbless

用于 HFS 和 HFS+ 文件的 Mac 式的 bless 工具

grub-menulst2cfg

将旧版的 GRUB menu.lst 转换为用于 GRUB 2 的 grub.cfg

grub-mkconfig

生成一个 grub 配置文件

grub-mkimage

创建 GRUB 可启动镜像

grub-mklayout

生成一个 GRUB 键盘布局文件

grub-mknetdir

准备一个 GRUB 网络启动目录

grub-mkpasswd-pbkdf2

生成一个加密的用于启动菜单的 PBKDF2 密码

grub-mkrelpath

将系统的路径名设为与其根目录相关

grub-mkrescue

创建适用于软盘或 CDROM/DVD 的 GRUB 可启动镜像

grub-mkstandalone

生成一个独立镜像

grub-ofpathname

用于打印 GRUB 设备路径的帮助程序

grub-probe

探测指定路径或设备的设备信息

grub-reboot

设置仅下次启动有效的 GRUB 默认启动项

grub-render-label

为 Apple Macs 调整 Apple .disk_label

grub-script-check

检查 GRUB 配置脚本的语法错误

grub-set-default

设置 GRUB 默认启动项

grub-sparc64-setup

grub-setup 的帮助程序

grub-syslinux2cfg

将 syslinux 配置文件翻译成 grub.cfg 格式
