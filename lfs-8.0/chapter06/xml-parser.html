6.41. XML::Parser-2.44

XML::Parser 模块是 James Clark 的 XML 解析模块 Expat 的 Perl 接口。

预计构建时间: 少于 0.1 SBU
需要磁盘空间: 2.0 MB

6.41.1. XML::Parser 安装过程

准备编译 XML::Parser: 

perl Makefile.PL

编译软件包: 

make

测试编译结果，输入: 

make test

安装软件包: 

make install

安装的模块: Expat.so
