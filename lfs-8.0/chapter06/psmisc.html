6.26. Psmisc-22.21

Psmisc 软件包包含用于显示进程信息的程序。

预计构建时间: 少于 0.1 SBU
所需磁盘空间: 4.0 MB

6.26.1. 安装 Psmisc

准备编译 Psmisc 软件包: 

./configure --prefix=/usr

编译该软件包: 

make

该软件包不含测试套件。

安装该软件包: 

make install

最终，将killall 和 fuser 程序移至 FHS 指定的位置: 

mv -v /usr/bin/fuser   /bin
mv -v /usr/bin/killall /bin

6.26.2. Psmisc 的内容

安装的程序: fuser, killall, peekfd, prtstat, pstree, and pstree.x11 (link to pstree)

简要描述

fuser

报告正在使用特定文件或者文件系统的进程ID(PIDs)

killall

杀死指定名称的进程; 它向所有名称相符的进程发送一个信号

peekfd

查看运行过程的文件描述符，给出了它的PID

prtstat

打印进程信息

pstree

以树形结构显示正在运行的进程

pstree.x11

与 pstree命令类似, 但是该命令会在退出前等待确认
