5.21. File-5.30

File 软件包包含用来判断文件类型的工具。

大概编译时间： 0.1 SBU
所需磁盘空间： 16 MB

5.21.1. 安装 File

准备编译 File： 

./configure --prefix=/tools

编译软件包： 

make

现在编译已经完成了。就像之前说的一样，不要求为本章中所构建的临时工具运行测试套件，如果你仍想运行的话，可以执行： 

make check

安装软件包： 

make install

该软件包的详细信息在 Section 6.12.2, “Contents of File.”
