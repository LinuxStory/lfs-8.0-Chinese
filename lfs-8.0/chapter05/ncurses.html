5.15. Ncurses-6.0

Ncurses 软件包包含了不依赖终端的处理字符界面的库。

大概编译时间： 0.5 SBU
所需磁盘空间： 38 MB

5.15.1. 安装 Ncurses

首先，确认 gawk 在配置时可以被找到： 

sed -i s/mawk// configure

准备编译 Ncurses： 

./configure --prefix=/tools \
            --with-shared   \
            --without-debug \
            --without-ada   \
            --enable-widec  \
            --enable-overwrite

configure 选项含义:

--without-ada

这个选项会保证 Ncurse 不会编译对宿主机系统里可能存在的 Ada 编译器的支持，而这在我们切换 chroot 环境后就不再可用。

--enable-overwrite

这个选项会告诉 Ncurses 安装它的头文件到 /tools/include 目录， 而不是 /tools/include/ncurses 目录， 这保证其他软件包可以成功找到 Ncurses 的头文件。

--enable-widec

这个选项会控制编译宽字符库（比如，libncursesw.so.6.0） 而不是一个普通库（比如，libncurses.so.6.0）。 这些宽字符库在多字节和传统的 8 位环境下都能使用，而普通库只能用于 8 位环境。宽字符库和普通库的源代码是兼容的，但并不是二进制兼容。

编译软件包： 

make

这个软件包有测试套件，但是只有在安装后才能执行。测试用例在 test/ 目录里。 查看该目录下的 README 文件以了解更多细节。

安装软件包： 

make install

该软件包的详细信息在 Section 6.20.2, “Contents of Ncurses.”
