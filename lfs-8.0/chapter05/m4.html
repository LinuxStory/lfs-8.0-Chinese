5.27. M4-1.4.18

M4 软件包包含了一个宏处理器。

大概编译时间： 0.2 SBU
所需磁盘空间： 19 MB

5.27.1. 安装 M4

准备编译 M4： 

./configure --prefix=/tools

编译软件包： 

make

现在编译已经完成了。就像之前说的一样，不要求为本章中所构建的临时工具运行测试套件，如果你想运行的话，可以执行： 

make check

安装软件包： 

make install

该软件包的详细信息在 Section 6.28.2, “Contents of M4.”
