5.17. Bison-3.0.4

Bison 软件包包含了一个生成语法分析器的程序。

大概编译时间： 0.3 SBU
所需磁盘空间： 32 MB

5.17.1. 安装 Bison

准备编译 Bison： 

./configure --prefix=/tools

编译： 

make

如果要测试结果，执行： 

make check

安装软件包： 

make install

该软件包的详细信息在 Section 6.29.2, “Contents of Bison.”
