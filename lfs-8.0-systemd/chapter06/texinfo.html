6.69. Texinfo-6.3

Texinfo 软件包包含阅读、撰写、转换信息页的程序。

构建约需时间： 0.5 SBU
需要磁盘空间： 108 MB

6.69.1. 安装 Texinfo

准备编译 Texinfo： 

./configure --prefix=/usr --disable-static

配置选项的含义：

--disable-static

在此情况下，顶级配置脚本不能识别这个选项，但是 XSParagraph 的脚本能识别此选项，并屏蔽安装静态 XSParagraph.a 到 /usr/lib/texinfo目录。

编译软件包： 

make

用以下命令测试结果： 

make check

安装软件包： 

make install

可选地，安装从属于 TeX 安装的组件： 

make TEXMF=/usr/share/texmf install-tex

make 指令参数的含义：

TEXMF=/usr/share/texmf

TEXMF 生成文件变量保存了 TeX 程序要被安装到的目标目录。

Info 文档系统使用纯文本文档来记录其菜单条目。此文件在 /usr/share/info/dir 。不幸的是，由于有的软件包的 Makefile 的问题，此目录有时不能与系统已安装的信息页同步。如果不需要重建 /usr/share/info/dir 文件，下面的可选命令将完成这个任务： 

pushd /usr/share/info
rm -v dir
for f in *
  do install-info $f dir 2>/dev/null
done
popd

6.69.2. Texinfo 的内容

安装的程序： info, install-info, makeinfo (link to texi2any), pdftexi2dvi, pod2texi, texi2any, texi2dvi, texi2pdf, and texindex
安装的库： XSParagraph.so
安装目录： /usr/share/texinfo and /usr/lib/texinfo

简要描述

info

读取和 man 页面相似的信息页，但通常比仅解释所有可用的命令行选项更加深入 [ 例如，比较 man bison 和 info bison。]

install-info

安装 info 页面；它会更新条目到 info 索引

makeinfo

将所给的 Texinfo 源文档转换为 info 页面，纯文本，或 HTML

pdftexi2dvi

将所给的 Texinfo 源文档转换为 PDF 文件

pod2texi

将 Pod 转换为 Texinfo 格式

texi2any

将所给的 Texinfo 源文档转换为其他格式

texi2dvi

将所给的 Texinfo 源文档转换为可打印的 div 格式

texi2pdf

将所给的 Texinfo 源文档转换为 PDF 文件

texindex

排序 Texinfo 索引文件
