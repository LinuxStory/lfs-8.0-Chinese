6.34. Bc-1.06.95

Bc 软件包，包含一个处理任意精度数值的语言。

预计构建时间： 0.1 SBU
所需磁盘空间： 3.6 MB

6.34.1. 安装 Bc

首先，修复代码中一些小的内存泄漏问题： 

patch -Np1 -i ../bc-1.06.95-memory_leak-1.patch

准备编译 Bc： 

./configure --prefix=/usr           \
            --with-readline         \
            --mandir=/usr/share/man \
            --infodir=/usr/share/info

配置选项的含义：

--with-readline

此选项让 Bc 使用系统中已安装的 readline 库，而不是使用它自带的版本。

编译软件包： 

make

如要测试 bc，运行下面的命令。输出会有点多，因此你可能会希望把它重定向到一个文件。测试中，最后一位数字会发生小概率的舍入误差（12144 次中可能会发生 10 次）。 

echo "quit" | ./bc/bc -l Test/checklib.b

安装软件包： 

make install

6.34.2. Bc 包含内容

安装的程序： bc 和 dc

简介

bc

是一个命令行计算器

dc

是一个反向修饰（运算符写在参数的最后面，一般是数字的最后面）的命令行计算器
