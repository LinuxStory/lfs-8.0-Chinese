5.16. Bash-4.4

Bash 软件包包含 Bourne-Again SHell。

大概编译时间： 0.4 SBU
所需磁盘空间： 61 MB

5.16.1. 安装 Bash

准备编译 Bash： 

./configure --prefix=/tools --without-bash-malloc

configure 选项含义：

--without-bash-malloc

这个选项会禁用 Bash 的内存分配功能（malloc），已知这个功能会导致段错误。而禁用这个功能后，Bash 将使用 Glibc 的 malloc 函数，这样会更稳定。

编译软件包： 

make

现在编译完成了。就像之前说的那样，现在不要求执行测试套件，如果仍想执行的话，可以运行： 

make tests

安装软件包： 

make install

创建一个从 bash 到 sh 的链接： 

ln -sv bash /tools/bin/sh

该软件包的详细信息在 Section 6.33.2, “Contents of Bash.”
