5.14. Check-0.11.0

Check 一个 C 的单元测试框架。

大约编译时间: 0.1 SBU
所需磁盘空间： 11 MB

5.14.1. 安装 Check

准备编译 Check： 

PKG_CONFIG= ./configure --prefix=/tools

configure 参数的含义：

PKG_CONFIG=

这个变量会告诉 configure 脚本忽略 pkg-config 里的任何可能会导致系统去链接 /tools 目录之外的库的选项。

构建软件包： 

make

现在编译已经完成了。就像之前说的一样，不要求为本章中所构建的临时工具运行测试套件，如果你想运行的话，可以执行： 

make check

注意 Check 的测试套件可能会耗费相对较长的时间（高达 4 SBU）。

安装软件包： 

make install

5.14.2. Check 软件包的内容

安装的程序： checkmk
安装的库： libcheck.{a,so}

简介

checkmk

用来生成 C 语言单元测试用例的 Awk 脚本，这些用例可以配合 Check 单元测试框架使用。

libcheck.{a,so}

包含允许测试程序调用 Check 的函数。
