5.19. Coreutils-8.26

Coreutils 软件包包含一套用于显示和设定基本系统属性的工具。

大概编译时间： 0.6 SBU
所需磁盘空间： 136 MB

5.19.1. 安装 Coreutils

准备编译 Coreutils： 

./configure --prefix=/tools --enable-install-program=hostname

配置选项含义：

--enable-install-program=hostname

这个选项会允许编译和安装 hostname 程序 – 默认是不安装的但是 Perl 测试套件需要它。

编译软件包： 

make

现在编译完成了。就像之前说的那样，现在不要求执行测试套件，如果仍想执行的话，可以运行： 

make RUN_EXPENSIVE_TESTS=yes check

RUN_EXPENSIVE_TESTS=yes 参数告诉测试套件额外运行一些对某些系统相对来说开销大一些（主要是 CPU 运算能力和内存消耗）的测试，但通常对 Linux 来说不是问题。

安装软件包： 

make install

该软件包的详细信息在 Section 6.51.2, “Contents of Coreutils.”
