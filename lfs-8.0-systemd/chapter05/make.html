5.28. Make-4.2.1

Make 软件包包含了一个用来编译软件包的工具。

大概编译时间： 0.1 SBU
所需磁盘空间： 12.5 MB

5.28.1. 安装 Make

准备编译 Make： 

./configure --prefix=/tools --without-guile

配置选项含义：

--without-guile

这个选项确保 Make-4.2.1 不会链接宿主系统上可能存在的 Guile 库，这些库在下一章的 chroot 切换环境后就不再可用。

编译软件包： 

make

现在编译已经完成了。就像之前说的一样，不要求为本章中所构建的临时工具运行测试套件，如果你想运行的话，可以执行： 

make check

安装软件包： 

make install

该软件包的详细信息在 Section 6.62.2, “Contents of Make.”
